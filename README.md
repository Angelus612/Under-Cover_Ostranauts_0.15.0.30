This is a mod (and a lite version of said mod) for Ostranauts, working in version 0.15.0.30 and is made by Angelus.
This mod allows all coveralls to be able to be worn underneath EVA suits, but not pressure suits due to clipping issues (This is the only thing the lite version does).
This mod disallows wearing the Ayotimiwa hoodie and grey port shirt underneath the brown duster due to clipping issues.
This mod gives the gunmetal and yellow coveralls additional inventory slots, in order to match the other coveralls.
This mod alters the loot table for NPC equipment, making it so they spawn with coveralls underneath EVA suits.
This mod gives the purple coverals an equivalent storage space to cargo (not harness) pants, due to its rarity.

After installing the mod, simply add "Under-Cover" to the "loading_order.json" file, or "Under-Cover-Lite" if you want to use the lite version.

PLEASE NOTE: If using the full mod, pre-existing gunmetal, yellow and purple coveralls will not get the expanded inventory, and will in-fact become bugged and lose some inventory. Coveralls currently present in shops still count as pre-existing. If you're currently storing anything in gunmetal, yellow or purple coveralls, please remove any items from them before installing the mod. You may spawn a replacement coverall using the debug menu after installing the mod.
