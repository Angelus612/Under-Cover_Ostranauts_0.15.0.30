This is a mod for Ostranauts, working in version 0.15.0.30 and is made by Angelus.
This mod allows all coveralls to be able to be worn underneath EVA suits (but not pressure suits due to clipping issues).
This mod disallows wearing the Ayotimiwa hoodie and grey port shirt underneath the brown duster due to clipping issues.
This mod gives the gunmetal coveralls additional inventory slots, in order to match the other tier2 coveralls.
This mod alters the loot table for NPC equipment, making it so they spawn with coveralls underneath EVA suits.
